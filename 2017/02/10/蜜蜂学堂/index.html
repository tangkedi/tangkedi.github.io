网易云案例分析

发表于 2017-02-10 | 分类于 校园经历 |

  小组成员 ：雷勋陶 汤迪 杜春鹏

产品简介

  网易云课堂，是网易公司打造的在线实用技能学习平台，该平台于2012年12月底正式上线，主要为学习者提供海量、优质的课程，用户可以根据自身的学习程度，自主安排学习进度。
  立足于实用性的要求，网易云课堂与多家教育、培训机构建立合作，课程数量已达4100+，课时总数超50000,涵盖实用软件、IT与互联网、外语学习、生活家居、兴趣爱好、职场技能、金融管理、考试认证、中小学、亲子教育等十余大门类。

战略总结

  从偏重实用技能学习的网络云课堂开始，与偏重人文社科及基础概念的网易公开课联合，打造“网易大教育品牌”，构建整体教育体系，一步一步抢占中国网络教育市场份额。

应用模式分析 Application model analysis

目标用户

  网易云课堂致力于打造为用户提供个性化网络教学服务、最实用的专业技能课程和最贴心有效的移动学习体验的在线实用技能学习平台。它的主要目标是为学习者提供海量、优质的课程，同时让用户可以根据自身的学习程度，自主安排学习进度。
  网易云课堂的目标用户是需要学习实用技能、考试升级的人群，比如即将大学毕业的学生、初入职场的新人或者是希望拓展工作技能的职场人士。

服务模式
基础功能

订阅：网易云课堂是一款个性化的线上学习软件，支持用户自主选课。当用户找到自己所需要的课程时，即可点击课程首页的“参与该课程”按钮。课程会自动添加到“我的学习”板块中，用户可以随时查找与管理。
评价：网易云课堂允许用户发表自己对于所学课程的看法和评价。评价采取星级制度，五星为最高评价，用户评价的内容可以随时更改没有限制。同时用户不能够通过评价与其他参与者发生互动。
分享：网易云课堂支持用户将课程一键分享到易信、易信朋友圈、微信、微信朋友圈以及微博、QQ和QQ空间这些主流社交平台。

特色功能

课程缓存：针对手机端移动便携的特点，网易云课堂提供了课程视频缓存功能。用户可以在连接wifi时缓存课程视频，以便随时随地都可以学习已经下载好的课程而不用受到网络、流量等方面的限制。免费课程只要参加学习就可下载，付费课程需要提交相关费用之后才可下载。
进度管理：针对已经学习过的课程，网易云课堂提供课程学习标记功能，允许用户自行标记是否已经完成该门课程。标记的结果会在“我的学习”主页面中显示。
微专业：网易微专业由网易云课堂联合各领域知名专家，以就业为导向，直达企业具体刚需岗位而精心打造的职业培训方案，解决传统教育与社会需求脱节的问题，学员可以快速、全面地掌握相关技能，并获得工作机会。同时参与者按要求完成学习后，考试通过即可获得相关专业认定证书。

服务流程

  用户登陆后可以查看“账号”板块中我的优惠券、我的购物车以及我的订单等类容，同时可以在我的学习板块管理已经下载及正在学习的内容。
  用户可以根据免费课程、热门课程、IT/互联网、职场提升、兴趣生活、语言留学、中小学等内容板块进行浏览，也可以直接搜索相关的关键词以定位感兴趣的内容。网易云课堂的课程内容分为免费课程和收费课程，其中收费课程也有部分免费试看的内容。用户可以把自己喜欢的课程下载到本地缓存并自动加入到参与的课程板块。

盈利模式

  1.网易云课堂收取一定的平台中介费。依托网易云课堂的流量资源，进而打造起来的网易云课堂会是在线教育平台的身份。网易云在这种课程上不单独收取费用，只提供平台，不生产内容。
  2.网易云课堂推出自己制作的精品课程，收取的费用完全归入自己门户。即是目前他主要推荐的“微课堂”。如网易旗下的网易杭州研究院，在网易云课堂上发布了一系列前端，后台，产品运营，计算机基础等各方各面的课程，收费在3000元左右浮动。
  目前在线教育的商业前景依然很值得发掘，比如去年网易云课堂推出以MOOC产品形态展现的“计算机专业课程”，上线一个月，截止12月1日开课前，便已累计收到了22万选课人次，这份火热也验证了在线教育的发展空间。网易云自己开设的精品课程的选课人数也可圈可点，可见研发课程也是其很明智的一个决定。

用户管理模式

  网易云课堂的用户只分为两类，即注册用户和浏览用户，浏览用户的权限很少，甚至不能够观看课程视频，所以其更多的是通过丰富的内容和完善的用户体验来吸引用户。网易云课堂可以通过网易邮箱、QQ、微信、微博和人人网等社交媒体来登陆账号，这样一方面可以免除用户重新注册账号的过程，减少用户的工作量，同时还可以通过这些社交媒体进行分享、发布以及与好友之间的互动。

运营状况管理分析

整体信息架构

用户体验分析————功能篇

  1.没有个性化推荐，版面全是系统推荐的内容


  2.课程评论区没有回复功能，评论质量低，没有“学习圏”的感觉


  3.学习过程中没有标记功能

  4.ios端不能找到已下载功能


  5.基本没有师生交流，沟通渠道很狭窄

用户体验分析————细节篇

  1.搜索功能差劲，经常发生错误


  2.视频不能锁定&安卓端视频不能直接横屏


  3.视频点击之后直接播放

产品推广模式

  在产品初期，网易云课堂在推广这一块很疲软，内部人员也曾说明基本是靠兄弟产品网易公开课的推广。随着网易云课堂慢慢的发展，它也慢慢开始通过赞助/举办大赛来推广自己的品牌。但是目前来看，网易云课堂的推广依然是个问题。如何找到更为合适有效的推广模式，对它来说也是亟待解决的。

结论与建议

  对于网易公开课的移动端来说，问题主要还是体现在互动方面。师生之间，用户之间，都没有足够的交流。社交性比较差。对于一个学习平台而言，如果没有与他人的讨论交流作为基础，就不能营造一种良好的互动学习氛围。除去交互上以及小功能上的区别，云课堂主要还是要在社交，沟通上下功夫。另外产品的推广也可以适当的加强。

创新

  蜜蜂学堂—-展示你的“蜂芒”！

目标定位

  网易云课堂致力于打造为用户提供个性化网络教学服务、最实用的专业技能课程和最贴心有效的移动学习体验的在线实用技能学习平台。
它的主要目标是为学习者提供海量、优质的课程，同时让用户可以根据自身的学习程度，自主安排学习进度。
  网易云课堂的目标用户是需要学习实用技能、考试升级的人群，比如即将大学毕业的学生、初入职场的新人或者是希望拓展工作技能的职场人士。

服务模式

  基础功能
  学习选课：蜜蜂学堂
是一款个性化的线上学习软件，支持用户自主选课。用户可以通过点击“参与课程”来进入学习自己所选择的课程。同时课程会自动添加到“我的学习”板块中，用户可以在该板块随时查找与管理。
  烂笔头（笔记）：蜜蜂学堂支持用户在学习视频课程时记录下自己的学习心得及体会。为了能够很好的同步用户笔记与具体课程的关系，笔记部分还包含有视频时间点等基础信息。同时，用户笔记分为私人笔记与公开笔记两部分，公开笔记在课程板块下，允许所有用户自由浏览；而私人笔记则只有用户本人可见。
  评价：蜜蜂学堂允许用户发表自己对于所学课程的看法和评价。评价采取星级制度，五星为最高评价，同时评价内容有着严格的限制，对于没有实际意义的评价将会定期删除。用户一旦对课程做出评价就不可更改。
  课后讨论：基于蜜蜂学堂做教育的本质，讨论模块也是十分重要的。讨论模块分为课程讨论区和比赛讨论区，课程讨论区主要提供关于在线视频课程内容的讨论，用户可以把自己在学习过程中的问题提出来，与其他学习者形成积极交流的气氛，有利于学习效率的提高；而比赛讨论区则提供比赛相关事务咨询功能，形成比赛小圈子，帮助用户更好的去了解比赛内容及注意事项。
  好友功能：蜜蜂学堂支持用户在浏览评价、公共笔记等用户发布内容时，能够点击用户头像来添加好友。用户点击头像可以链接到个人空间页面，空间主页提供部分主人的介绍以及动态，同时还可以通过主页面的好友添加功能来向主人发送好友请求，当主人同意后用户即可拥有更高的浏览权限。

特色功能

  学习私人化定制：蜜蜂学堂提供学习私人化定制功能，它在每个大类的推荐模块里面。用户进入学习私人化定制后，会对自己想学的科目，自身的基础，想学到的程度作出选择，然后我们根据用户的个人情况进行课程推荐，达到学习私人化定制的目的。
  自定义学习模式：用户可以选择自由学习模式和计划学习模式。在自由学习模式中，用户可以自由选择课程学习，而在计划学习模式中，将会启用我们的“蜂鸣系统”，我们将为用户制定学习计划，并每天在用户设定的时间段进行提醒学习和复习。
  蜂芒大赛：我们每个月将在每个大板块开展一次比赛，比赛分为初级场和高机场，用户根据自己的学习水平选择。用户有半个月的准备时间，来参加本次比赛。同时在比赛的过程中看到他和他同时学习的同学之间优势与不足。大赛还会有专门的评委点评自己选出的前十作品，看看业界人士对于作品的看法。比赛奖励一定数额的奖金，前几名能够提供推荐给合作公司的机会。每月优秀作品会整理成册，以供参考。
  特色成就系统：用户完成了一门课程，会获得成就认证。参加蜂芒大赛获得名次也会
一定奖励认证。完成课程以及打卡到一定次数会有奖励。认证可以分享到你的社交平台。并且获得某些特定荣誉可以获得荣誉头像框，彰显身份

产品盈利模式

  我们的盈利主要有三种方式大赛赞助、作品出版、平台抽成。
  大赛赞助，我们提供的题目可以是赞助方自己命题，来获得相应的创意来源，利用大赛制造社会热点吸引关注的软广告，发现对自己公司有利的人才提前拉拢，在未入职场或者刚入职场的人心中建立良好地位。
  作品出版，在参赛条例中写明会有出版使用权后进行整理出版，以电子书的形式出版。
  平台抽成，对于收费的课程我们会以一定比例收取平台管理费。

用户管理模式

  1.注册认证管理。用户可以通过新浪微博、微信号、邮箱、手机号、蜂号（独立注册管理账户）认证登录。
  2.用户分类管理模式。蜜蜂学堂的用户包括一般用户（学员）、内容提供者（课程生产者）。用户是消费信息同时产生有意义的讨论与作品，而内容提供者则提供课堂。
  3.用户激励管理模式。蜜蜂学堂有完善的积分系统和成就认证系统。
  积分系统是根据你的学习程度来奖励一定积分，换取一些特权。在你学习过程中，你的问答有百人以上认同，会有积分奖励。比赛的名次获取，我们会给予相应的积分奖励。你完成了一门课程，根据完成度来给予积分。平时打卡签到到一定的次数也可以获得积分。积分可以兑换不同的头像框与徽章奖励。
  成就认证系统是，你完成课程，或者在比赛中获得荣誉，有较大的贡献，我们会提供一些证书认证来分享到社交平台中去。

  后记:目前的在线教育平台多只是一个”平台”，而不是一个圈子，用户来了只是单纯的学习，没有学习的规划与学习的热情，而我们以蜜蜂为喻，同样以蜜蜂的勤劳为出发点设计应用，在线教育也应该是有目标，有交流，有讨论的。于是就产生了“蜂芒毕露”，成就系统，比赛讨论区这样的功能设计，希望能够用这些东西激起学习者们的热情。
